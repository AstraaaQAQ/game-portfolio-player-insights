# Notebooks

The notebooks in this folder have been cleared of cell outputs for public sharing.

Each notebook starts with a short method note covering the research question, input data, filters, output tables, and limitations. The notebooks show the analysis workflow used to collect, clean, tag, aggregate, and prepare data for Power BI dashboards.

Raw review-level datasets are excluded from this repository. Aggregated outputs used in the public portfolio are available in `data/`.
